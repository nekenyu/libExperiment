Detailed Description

template<typename T>
class Experiment::DoubleLinkedListImpl::Node< T >

DoubleLinkedList internal Node base class.

This is a common implementation with DataNode which adds the data value.

DoubleLinkedList and/or Iterator is responsible for allocation, deallocation and management of a Node's next and previous nodes. Node simply provides the ability to set these pointers.

Template Parameters
Tthe type of the value this Node works on
Note
This class is not accessible to the users of DoubleLinkedList.
